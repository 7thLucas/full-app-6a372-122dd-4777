# ForgeRealms — Product Brief

## What It Is
An AI-powered, no-code game development platform for the multiplayer strategy genre. Indie creators pick a map template, customize their factions and world, and generate a fully playable multiplayer strategy game — no code, no team, no $2M budget.

## Target Users
- Primary: Indie creators with vision but no technical skills
- Secondary: Aspiring first-time game developers
- Tertiary: Small studios prototyping strategy titles fast

## Core Problem
Building a real multiplayer strategy game (Mobile Strike / Kings of Avalon / Throne Kingdom at War style) requires 10–50 person teams, $500K–$5M, and 2–5 years. The tools are too complex; big studios intentionally keep the barrier high.

## Core Solution
1. Genre-specific map templates modeled on Mobile Strike, Kings of Avalon, Throne Kingdom at War
2. Orchestrated AI toolchain: world gen, unit design, event scripting, balance tuning, multiplayer server setup
3. Visual conversational interface — so simple a 5-year-old can use it
4. Ships real deployable multiplayer games with persistent state, real-time interaction, alliances, resources, combat

## First Feature (MVP)
Pick a map template → customize factions → hit Generate → receive a live playable multiplayer strategy game

## Brand
- Name: ForgeRealms
- Tone: Empowering, bold, creator-first — speaks to the dreamer, not the engineer
- Tagline: Build a real multiplayer strategy game. No code. Just imagination.

## Strategic Principles
- Real over fake: every output is a deployable game, never a demo
- Genre-authentic: templates grounded in proven mechanics players know
- AI-first: every step has an AI agent removing friction
- One source: concept to deployed game, no third-party tools
- Child-simple UX: if a 5-year-old can't navigate it, the interface fails
